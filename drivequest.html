Let's Work Together

Project : DriveQuest

December 2023 - Present

DriveQuest is an interactive and educational grid-based trivia game focused on driving rules and road safety. Developed as part of a group project, the game aims to provide a fun and engaging way for players to test their knowledge of driving regulations.

My Contribution

Figma Design Collaboration:

Worked closely with the team on the design phase using Figma. Contributed to the layout, color scheme, and user interface elements to ensure an intuitive and visually appealing experience.

Backend Functionality:

Took the lead on developing the backend functionality of the game. Implemented server-side logic using Node.js and Express.js to handle API requests and manage game data.

Database Integration:

Integrated MongoDB to store and retrieve questions, player data, and game progress. Designed the database schema to support the game's requirements and ensure efficient data retrieval.

Game Logic:

Developed the core game logic, including question selection, turn-based mechanics, and scoring system. Ensured a smooth and error-free gameplay experience.

Technologies Used

Frontend: HTML, CSS, JavaScript

Backend: Node.js, Express.js

Database: MongoDB

Design: Figma

Version control: GitHub
